# BikeRentApp
Greetings. This application is intented to simulate bicycle renting system.

The technologies used:
- Frontend
  - HTML5
  - CSS3
  - Bootstrap
  - Angular 13
  
- Backend:
  - ASP.NET API
  -Entity Framework Core
  
- Database:
  - MS SQL Server
  
  

## Requirements for testing:
- .NET 5
- Visual Studio 2022
- [npm (node.js)](https://nodejs.org/)
- Microsoft SQL Server (LocalDb)
